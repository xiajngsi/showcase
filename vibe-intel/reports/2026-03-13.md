## 二、新产品与技术动态

**Superpowers** ｜ Agent 技能框架 ⭐ 81,101 stars
- **是什么**：一套完整的 AI 编程代理工作流方法论，通过可组合的"技能"模块驱动编码代理
- **核心功能**：技能编排 / 工作流标准化 / 多代理适配
- **目标用户**：重度使用 AI 编程的开发者
- **为何值得关注**：8 万+ stars 说明社区对"如何系统化使用 AI 编程"的强烈需求
- 🔗 [GitHub](https://github.com/obra/superpowers)

**Anthropic Claude Plugins 官方目录** ｜ 插件生态 ⭐ 10,568 stars
- **是什么**：Anthropic 官方维护的 Claude Code 高质量插件目录
- **核心功能**：插件发现 / 质量审核 / MCP 协议集成
- **为何值得关注**：标志着 Claude Code 从工具走向平台，生态护城河加速构建
- 🔗 [GitHub](https://github.com/anthropics/claude-plugins-official)

**Hermes Agent** ｜ 自进化 AI Agent ⭐ 6,516 stars
- **是什么**：Nous Research 出品的具备学习循环能力的 Agent——能从经验中生成技能、跨会话积累用户理解
- **核心功能**：自动技能学习 / 持久化记忆 / 多模型支持 / 可部署在 $5 VPS
- **为何值得关注**：首个将"自我改进"作为核心卖点的开源 Agent，指向 Agent 竞争的下一阶段
- 🔗 [GitHub](https://github.com/NousResearch/hermes-agent)

**Mozzie & KingCoding** ｜ 多 Agent 编排工具
- **是什么**：分别支持 CLI 和 Dashboard 方式并行运行多个 AI 编程代理
- **为何值得关注**：验证了"开发者需要同时管理多个 AI 代理"这一新需求的真实性
- 🔗 [Mozzie](https://www.producthunt.com/products/mozzie) / [KingCoding](https://www.producthunt.com/products/kingcoding)

**Axe** ｜ 轻量级 AI 框架（HN 195 分）
- **是什么**：12MB 单一二进制文件，号称替代臃肿的 AI 框架
- **为何值得关注**：113 条评论反映开发者对当前 AI 框架复杂性的强烈不满——简洁化是真需求
- 🔗 [GitHub](https://github.com/jrswab/axe)

**分析师点评：** **本日产品信号高度一致——行业正从"单一 AI 编程助手"快速演进到"多 Agent 编排 + 插件生态 + 自进化"的复合架构阶段。**

## 五、分析师判断

**本周趋势方向：** AI 编程工具竞争正式从"单点效率"进入"编排层 + 生态层"竞争。多 Agent 并行、插件市场、自进化能力成为新的差异化维度。Shopify CEO 亲自用 AI 做性能优化（[Liquid PR 提升 53% 性能](https://simonwillison.net/2026/Mar/13/liquid/#atom-everything)）则预示：C-level 亲自上手 vibe coding 将成为新常态。

**今日最强信号：** Anthropic 推出官方插件目录（10K+ stars），Claude Code 正式从工具转向平台——这是生态锁定的起手式。

**需持续观察：** 多 Agent 编排工具（Mozzie/KingCoding/Superpowers）的实际留存率——当前爆发可能是需求验证，也可能是昙花一现的工具碎片化。
